# Brand Guide — Jungle Nidra

Consolidated visual/audio identity reference so every video pulls from the same base
instead of re-deriving it each time. Voice/narration and CapCut editing settings live
in `PRODUCTION-SOP.md` (Parts 2-3) — not duplicated here, that file is the source of
truth for those.

## Color palette

Pulled from the pattern used in both thumbnail concepts built so far
(`02-packaging/thumbnails/`): 3 colors max per thumbnail, always from this set —

- **Deep blue** — night sky, deep water, the "calm/vast" end of the palette
- **Soft moss green** — jungle, temple stone, the "grounded" end of the palette
- **Warm low amber glow** — the one warm accent (candlelight, moonlight on water,
  soft firelight) — never more than one warm light source per thumbnail

Muted/night-friendly tones only. No harsh saturation, no colors typical of normal
"clickbait" YouTube thumbnails.

## Imagery style

Nature/ambient/symbolic — never a human face or figure, never Eli's identity, never a
named real-world location (see brand guardrails in every `jn-*` skill). Draws from the
location library (`04-recording-assets/location-library.md`): water, waterfall,
temple stone, jungle canopy. Soft light preferred — early morning/late afternoon
shoots, moonlit or low-light thumbnail concepts.

## Thumbnail construction template

Every thumbnail follows this same 4-part structure (from `jn-holy-trifecta` Step 2):
1. **VISUAL** — one calm, inviting scene, no face
2. **PALETTE** — 3 colors max, from the set above
3. **TEXT** — 3-4 words max, reinforces the promise, never alarmist (e.g. "ANXIETY
   RELIEF SLEEP", "FALL ASLEEP TONIGHT")
4. **FOCAL** — the single element that anchors the visual metaphor (a still pool, a
   glowing path, a candle)
